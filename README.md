## ✨ Features

- ➕ Basic arithmetic operations — Addition, Subtraction, Multiplication, Division
- 💯 Percentage calculation
- ⌫ Delete last input (backspace button)
- 🔄 Clear all input (AC button)
- 🔢 Decimal number support
- 📱 Responsive and clean UI
- 🎨 Custom Poppins font for a modern look
- ⚡ Real-time expression display with output preview

## 🚀 Getting Started

### Prerequisites
No installations required! Just a web browser.

### Run Locally

1. **Clone the repository**
   ```bash
   git clone https://github.com/Umairalimalik78/Web-Calculator.git
   ```

2. **Navigate to the project folder**
   ```bash
   cd Web-Calculator
   ```

3. **Open `index.html` in your browser**
   ```bash
   open index.html
   ```
   Or simply double-click the `index.html` file.

---

## 🎮 How to Use

1. Click number buttons to enter digits
2. Click operator buttons (`+`, `-`, `×`, `÷`, `%`) to choose an operation
3. Click `=` to calculate the result
4. Click `AC` to clear all input
5. Click the backspace button `⌫` to delete the last character
